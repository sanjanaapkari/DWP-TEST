# Book Library System UI Automation

This project contains end-to-end UI test automation for the Book Library System application using **Playwright** with **JavaScript**.

## Prerequisites

Make sure these are installed on your machine:

- [Node.js](https://nodejs.org/)
- npm

To verify installation:

```bash
node -v
npm -v

## Steps to clone and run the tests locally
1. Clone the repo - git clone <your-repository-url>
2. Navigate to the repo directory - cd <your-project-folder>
3. Install the dependencies - npm install
4. Install the playwright browsers - npx playwright install
5. To run all tests in headed mode - npx playwright test --headed
6. To run all tests in one browser (eg: Chromium) - npx playwright test --project=chromium --headed
7. To view the report - npx playwright show-report
